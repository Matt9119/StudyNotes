🔐 Permission Levels & NTFS Rules
FFull controlicacls file /grant Alice:F
MModifyicacls file /grant Alice:M
RXRead and executeicacls file /grant Alice:RX
RRead onlyicacls file /grant Alice:R
WWrite onlyicacls file /grant Alice:W
NTFS PRECEDENCE RULES
Explicit Deny
↓ beats everything
Explicit Allow
↓ beats inherited
Inherited Deny → Inherited Allow
Local access: NTFS perms only
Share access: Most restrictive of NTFS + Share perms wins
Group perms: Cumulative with user perms
🚨
Copy vs Move: Copying within NTFS creates a new file that inherits target folder perms. Moving within the same partition keeps original perms. Moving across partitions inherits target. Moving NTFS→FAT loses all permissions and attributes.

📂 Registry Hive Reference
Full NameAcronymTypePurpose & Notes

🔑 Important Registry Key Paths
Registry PathContains / Purpose
HKLM\SAMLocal account credentials (SAM database). Requires SYSTEM privileges to read directly.
HKLM\SECURITY\CacheDomain controller cached credentials.
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersionOS version, build number, install date.
HKLM\SYSTEM\CurrentControlSet\ServicesAll installed services and their configuration.
HKLM\SYSTEM\CurrentControlSet\Control\ComputerName\ComputerNameMachine hostname.
HKCU\Software\Microsoft\Windows\CurrentVersion\RunPrograms that run at login for the current user (common persistence location).
HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\RunPrograms that run at login for all users (common persistence location).
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\WinlogonLogon settings — Userinit, Shell values often abused for persistence.
HKCU\EnvironmentUser-level environment variables.
HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\EnvironmentSystem-wide environment variables including PATH.
🔑
Root hives: Only HKLM and HKU are true root hives stored on disk. HKCR, HKCU, and HKCC are linked hives — pointers to subkeys within the root hives. Structure: Keys (like folders) → Values (like files) → Data (content).
Process Management
⊡ CMD vs ⚡ PowerShell process commands
CMDPowerShellDescription + Example
tasklist Get-Process (ps / gps) List running processes. tasklist /svc shows hosted services per process.
tasklist /fi "PID eq 1234" Get-Process -Id 1234 Filter by PID.
tasklist /fi "IMAGENAME eq notepad.exe" Get-Process -Name notepad Filter by process name.
taskkill /PID 1234 Stop-Process -Id 1234 Kill by PID.
taskkill /IM notepad.exe Stop-Process -Name notepad Kill by image name.
taskkill /F Stop-Process -Force Force kill. taskkill /F /PID 1234
— Get-Process | Get-Member See all available properties and methods on process objects.
— Get-Process | Where-Object {$_.CPU -gt 10} Filter processes by property (CPU, WorkingSet, etc.).
— Get-Process | Select-Object Name,Id,CPU Isolate specific properties from process objects.
— Get-Process | Measure-Object Count total number of running processes.
🚨
Key difference: Get-Process returns objects with a PID property. Get-Service does NOT have a PID — use tasklist /svc in CMD or Get-WmiObject Win32_Service in PS to associate services with PIDs.
Service Management
⊡ CMD — sc & net vs ⚡ PowerShell — Get-Service
CMD (sc / net)PowerShellDescription + Example
sc query Get-Service List all active services. sc query uses service key name.
sc query type= all Get-Service | Format-List * All services including inactive ones.
sc query "wuauserv" Get-Service -Name wuauserv Query a specific service by name.
sc queryex "wuauserv" Get-Service wuauserv | Select * Extended status — shows PID and additional flags.
sc start "wuauserv" Start-Service -Name wuauserv Start a service by key name. net start "Windows Update" uses display name.
sc stop "wuauserv" Stop-Service -Name wuauserv Stop a service.
sc qc "wuauserv" Get-WmiObject Win32_Service -Filter "Name='wuauserv'" Query service configuration (binary path, start type, etc.).
sc config "svc" start= disabled Set-Service -Name svc -StartupType Disabled Change service start type.
sc getkeyname "Windows Update" — Get key name from display name — useful before using sc commands.
net start Get-Service | Where Status -eq Running net start with no args lists display names of all running services.
💡
sc vs net: sc start/stop uses the service key name. net start/stop uses the display name. Use sc getkeyname "Display Name" to convert between the two.
