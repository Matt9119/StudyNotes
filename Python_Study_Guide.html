Python Fundamentals
Intro to Python
Two key characteristics
Python is an interpreted language and an object-oriented programming (OOP) language.
PEP8
Python Enhancement Proposal 8 — the standard style guide for writing and formatting Python code. Follow it for proper, readable code.

Types of languages

TypeHow it worksPython?

InterpretedReads and executes each line one at a time✅ Yes

Building blocks of OOP

Classes
Data types that act as blueprints for objects, attributes, and methods.
Objects
Classes that have been created with defined data — an instance of a class.
Methods
Functions inside classes that describe the behaviors of an object.
Attributes
Defined in classes and represent the state of an object.

Running Python — 3 ways

Command Line
Execute a one-liner directly: python -c "print('hello world')"
Python Shell (Interactive)
Type python in the CLI to enter interactive mode. Great for testing small snippets.
Execute Scripts via CLI
Run .py or .pyc files directly: python script.py

Key takeaway

Python uses whitespace and indentation to define blocks of code — not braces. A block is a group of statements at the same indentation level. Inconsistent indentation will crash your script.

Variables & Data Types

Variable rules

  • Names must start with a letter or underscore — not a number.
  • Names are case-sensitive (myVar ≠ myvar).
  • Python automatically assigns a variable's type unless you explicitly cast it.
  • Code on the right side of = always executes first.
  • Python does not clean up old memory when a variable is reassigned — avoid storing sensitive data in variables.

Strings, Lists, Tuples, Sets & Dicts

Strings

Immutable, zero-indexed
First element is always index 0. Cannot be changed after creation.
Slicing
string[start:stop:step] — extract a substring.
.format()
Dynamically insert variables into a string: "Hello {}".format(name)
Escape characters
Backslash \ inserts special/illegal characters into strings.
.encode() / .decode()
Convert string → bytestring (.encode()) or bytestring → string (.decode()). Bytestrings are used to transfer data over networks.

Lists

Mutable, ordered, zero-indexed
Can contain any object type, including other lists (nested).
.append()
Add to the end. Cannot assign beyond the last index — must use this.
.insert(i, obj)
Insert at a specific index.
.split() / .join()
Convert string → list (.split()) or list → string (", ".join(list)).
map(func, list)
Apply a function to every element of a list or iterable.
zip(list1, list2)
Groups items at the same index into tuples. Drops extras if lists are unequal length.
Nested access
matrix = [[1,2,3],[4,5]] → matrix[0][1] = 2

Tuples, Sets & Dictionaries

Tuple
Immutable, ordered, zero-indexed. Used to "pack" a fixed group of values. More efficient than lists.
Set
Mutable, unordered, unique values only. Use set() for empty — not {}.
Dictionary
Mutable key:value pairs. Keys must be unique. Values can repeat. Very fast lookup.

Type casting

Explicit casting
Pass a variable to a type function: str(100) → "100"
Implicit casting
Python auto-casts in some situations — int + float = float, str + str = concatenate.
True copies
Variables are pointers to data. To copy a list/dict/set without linking to the original, use .copy().

Control Flow
Control Statements
What is control flow?
The order in which individual statements are executed and how decisions are made within a program.
3 types
Condition-based (run if true) · Count-based (run N times) · Collection-based (run over iterable)

Condition-based statements

if
Executes block if expression evaluates to True.
elif
Checked only if prior if/elif was False. Only the first True block executes.
else
Runs when all preceding conditions are False.
match / case
Concise pattern matching. case _: is the catch-all (like else). Only first matching case executes.
while
Conditional loop — executes as long as condition is True or until a break fires.
# if / elif / else pattern if x > 10: print("big") elif x > 5: print("medium") else: print("small")

Collection-based — for loop

for item in list
Steps through every element of a list or iterable.
enumerate(list)
Returns both the index and value on each iteration.
# enumerate — get index AND value for i, val in enumerate(my_list): print(i, val)

Count-based — range() & len()

range()
Generates a sequence of numbers. Used in for loops or standalone to create a list.
len()
Returns object's length. Useful when iterating by index.
List comprehension
Shorthand for building lists using for, if, and range() in one line.
# range in for loop for i in range(0, 10): print(i) # list comprehension squares = [x**2 for x in range(10)]

Nested loops & loop control

Nested loops
Inner loop runs completely for each single iteration of the outer loop.
break
Exits the closest enclosing while or for loop entirely.
continue
Skips the rest of the current iteration and jumps to the next one.
No do-while
Python has no do-while — simulate with a while True loop and break.
Functions
def keyword
Defines a function. Code only runs when the function is called (invoked).
Arguments
Passed in after the function name. Can be optional or have default values.
Local scope
Variables inside a function exist only inside it (local variables).
Global scope
Variables outside all functions are global. Modify inside a function with global varname — not recommended.
return
Best practice to use return to pass a local value up to a higher scope.
pass
Lets Python recognize a code block with no implementation yet (placeholder).
# basic function with return def add_numbers(a, b=0): # b has a default value result = a + b return result total = add_numbers(5, 3) # total = 8
Modules
What is a module
A collection of reusable functions and data structures that add additional functionality.
import
Load a module with import module_name.
__name__
Set to "__main__" when run directly, or the module name when imported. Use if __name__ == "__main__": to prevent a module from executing when imported.

collections module

defaultdict
Creates a dictionary with a default value for undefined keys — queries never raise a KeyError.
Counter
Creates a dictionary that automatically counts the number of times a key is used. Useful for frequency analysis and simple cipher decryption.

Proper script structure

  1. import statements (if any)
  2. Docstring and comments
  3. Function declarations (with docstrings)
  4. main() function declaration
  5. Check for main: if __name__ == "__main__":
  6. Global variable declarations (if any)
  7. Call to main()
Key takeaway

Importing a user-defined module executes it by default. Prevent this by checking if __name__ == "__main__": — the module's code only runs when called directly, not when imported.

File Manipulation & Data Analysis
User Input & Error Checking
Assume untrustworthy
User input should always be verified — unverified input can cause crashes.
Traceback
When Python hits an unhandled error, it crashes and prints a traceback showing the line number and error type.
try / except
Handle errors gracefully without crashing. Can catch specific error types or use a catch-all except.
# try / except pattern try: result = 10 / user_input except ZeroDivisionError: print("Cannot divide by zero") except TypeError: print("Wrong type") except: # catch-all print("Unknown error")
Reading & Writing Files

Opening files

Assign reader (manual close)
reader = open("file.txt") — must close manually with reader.close().
With statement (auto close)
with open("file.txt") as reader: — automatically closes when the block exits. Preferred.

File open modes

CharacterMode
'r'Open for reading (default)
'w'Open for writing — overwrites existing content
'a'Append — write to the end of an existing file
'rb' / 'wb'Binary read/write mode — uses byte data

Reading & writing functions

FunctionDescription
.read(size=n)Read n bytes from the file
.readline(size=n)Read n characters from a single line
.readlines()Read all remaining lines — returns each line as a list item
.write(string)Write a string to the file
.writelines(seq)Write a sequence (e.g. list of strings) — does NOT add newline characters
# read all lines from a file (preferred pattern) with open("data.txt", "r") as f: lines = f.readlines() for line in lines: print(line.strip())
Pro tip

Always use the with open() syntax — it guarantees the file is closed even if an error occurs inside the block. Forgetting to close files can cause data corruption or resource leaks.

OCO & DCO with Python
Networking — Scapy
What is Scapy
An extensive packet crafting module — sniff, read, write, craft, forge, and modify packets directly in Python.
PacketList
Scapy represents captured traffic as a PacketList object — structured and accessed like a Python list.
Layer access
PacketList[pkt_num][layer_name] — layer names are case-sensitive.
.haslayer()
pkt.haslayer("TCP") — check if a packet contains a given layer.
Looping
Use a for loop to iterate through all packets in a PacketList.

Case-sensitive Scapy layer names

Layer nameOSI layer
EtherLayer 2 — Ethernet
IPLayer 3 — IPv4
TCPLayer 4 — TCP
UDPLayer 4 — UDP
DNSApplication — DNS
RAWRaw payload / application data
# iterate through a Scapy PacketList from scapy.all import rdpcap packets = rdpcap("capture.pcap") for pkt in packets: if pkt.haslayer("TCP"): print(pkt["IP"].src, pkt["TCP"].dport)
Sockets
socket module
Enables connections and data transfers over IP using UDP or TCP. Supports both IPv4 and IPv6.
socket.SOCK_DGRAM
Identifies a UDP socket.
socket.SOCK_STREAM
Identifies a TCP socket — enables two-way communications.
socket.error
Common error caused by connection timeout.

TCP Client vs TCP Server

RoleFunctions usedPurpose
TCP Client.connect()Initiate the TCP 3-way handshake to a server
TCP Server.bind() → .listen() → .accept()Bind to port, listen for connections, accept incoming client
import socket # TCP client example s = socket.socket(socket.AF_INET, socket.SOCK_STREAM) s.connect(("192.168.1.1", 4444)) # TCP server example srv = socket.socket(socket.AF_INET, socket.SOCK_STREAM) srv.bind(("", 4444)) srv.listen(1) conn, addr = srv.accept()
Pro tip

On Linux, Netcat can simulate a socket server by setting up a listener: nc -lvp 4444. Useful for testing Python socket clients without writing a full server.

Subprocess Module
What it does
Spawns new system processes, connects to their stdin/stdout/stderr, and retrieves return codes.

Subprocess functions

FunctionSyntaxDescription
Popensubprocess.Popen(cmd)Spawns a process — allows you to see output while it runs
communicateproc.communicate()Returns (output, error) from a process's execution
callsubprocess.call(cmd)Spawns process, waits for it to finish, returns 0 (success) or non-zero (fail)
runsubprocess.run(cmd)Returns a process object — may not capture errors

Key subprocess arguments

ArgumentDescription
stdinCaptures input to a child subprocess
stdoutCaptures output from a child subprocess
stderrCaptures errors from a child subprocess
shellSpecifies if command runs through the system shell
subprocess.PIPEPipes stdin/stdout/stderr to .communicate()

import subprocess # run a command and capture output proc = subprocess.Popen( ["ls", "-la"], stdout=subprocess.PIPE, stderr=subprocess.PIPE, text=True ) output, error = proc.communicate() print(output)
Creating Python Executables
Goal
One OCO objective is establishing a foothold on a target system — a Python backdoor is one method.
Problem
A Python backdoor only works if the target has Python installed.
Solution — PyInstaller
Converts .py scripts to standalone Windows executables (.exe). No Python install required on the target.
HTTP Server
Python's built-in HTTP Server module can be used to transfer files across a network — useful for payload delivery or exfil.
Exam trap

Know the difference between OCO (Offensive Cyber Operations — establish foothold, backdoors, exfil) and DCO (Defensive Cyber Operations — detect, analyze, respond). Python's networking tools serve both sides depending on context and intent.
